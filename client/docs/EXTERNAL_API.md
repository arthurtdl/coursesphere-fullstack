# 🛠️ Justificativa de Uso

> Para o cumprimento do requisito de consumo de API pública externa, foi integrada a RandomUser API.  
> A escolha se deu pela sua estabilidade, conformidade com os padrões REST e pela riqueza de dados multimídia (fotos, nomes e localizações), permitindo validar o fluxo assíncrono do sistema e o tratamento de dados de terceiros sem a necessidade de chaves de acesso estáticas, garantindo a dinamicidade da interface.
